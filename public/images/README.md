# Guia de Organização de Imagens - Taipei Coffee House

## 📁 Estrutura de Pastas

```
public/images/
├── hero/          # Imagens principais do banner/hero
├── menu/          # Fotos de pratos, bebidas e comidas
├── brunch/        # Fotos específicas do brunch
├── eventos/       # Fotos do espaço de eventos e celebrações
├── ambiente/      # Fotos do ambiente interno e externo
└── sobre/         # Fotos para a página "Sobre"
```

## 📋 Como Organizar as Imagens Baixadas do Wix

### 1. **Hero/Banner** (`/hero/`)
Coloque aqui:
- Imagem principal do banner da home
- Fotos panorâmicas da vista para o mar
- Imagens de destaque do restaurante
- Sugestão de nomes: `hero-main.jpg`, `hero-vista-mar.jpg`

### 2. **Menu** (`/menu/`)
Coloque aqui:
- Fotos de cafés especiais
- Sanduíches e bruschettas
- Doces e tortas
- Pratos principais
- Bebidas
- Sugestão de nomes: `cafe-especial.jpg`, `bruschetta.jpg`, `torta-chocolate.jpg`

### 3. **Brunch** (`/brunch/`)
Coloque aqui:
- Fotos do brunch completo
- Pratos do brunch
- Mesa montada
- Sugestão de nomes: `brunch-completo.jpg`, `brunch-mesa.jpg`

### 4. **Eventos** (`/eventos/`)
Coloque aqui:
- Fotos do espaço de eventos
- Decorações de festas
- Eventos realizados
- Espaço kids
- Sugestão de nomes: `espaco-eventos.jpg`, `festa-aniversario.jpg`, `kids-area.jpg`

### 5. **Ambiente** (`/ambiente/`)
Coloque aqui:
- Fotos do interior do café
- Área externa
- Mesas e decoração
- Vista para o mar
- Sugestão de nomes: `interior.jpg`, `area-externa.jpg`, `vista-mar.jpg`

### 6. **Sobre** (`/sobre/`)
Coloque aqui:
- Fotos da equipe
- História do Taipei
- Fotos institucionais
- Sugestão de nomes: `equipe.jpg`, `fundadora.jpg`

## 🎨 Dicas de Otimização

Antes de adicionar as imagens:

1. **Redimensione** para tamanhos adequados:
   - Hero: 1920x1080px (máximo)
   - Cards/Grid: 800x600px
   - Thumbnails: 400x300px

2. **Otimize o tamanho**:
   - Use ferramentas como TinyPNG, Squoosh ou ImageOptim
   - Mantenha qualidade 80-85%
   - Converta para WebP quando possível

3. **Nomeie corretamente**:
   - Use nomes descritivos em minúsculas
   - Separe palavras com hífen: `cafe-especial.jpg`
   - Evite espaços e caracteres especiais
