# Taipei Coffee House - Site Oficial

Migração do site do Wix para Next.js com React e Tailwind CSS.

## 🚀 Estrutura do Projeto

### Tecnologias
- **Next.js 14** - Framework React com App Router
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização utilitária
- **Framer Motion** - Animações suaves e modernas
- **Lucide React** - Ícones modernos

### Páginas Criadas (do Wix)
- `/` - Home com hero animado e seções
- `/brunch` - Página do Brunch com menu completo
- `/menu` - Cardápio completo por categorias
- `/sobre-o-taipei` - História e valores da marca
- `/restaurante-em-cacupe` - Localização e informações
- `/cafeteria-com-espaco-kids-florianopolis` - Espaço Kids
- `/espaco-eventos-florianopolis` - Espaço para eventos
- `/service-page/eventos-corporativos-em-florianopolis` - Eventos corporativos
- `/service-page/eventos-sociais` - Eventos sociais

### Componentes
- `Header` - Navegação responsiva com menu mobile animado
- `Footer` - Rodapé com links e informações
- `AnimatedSection` - Wrapper para animações de scroll

## 🎨 Design System

### Paleta de Cores (Oficial)
```css
--taipei-red: #A84544      /* Cor principal */
--taipei-cream: #F8F1EB    /* Fundo suave */
--taipei-sage: #9AAB9E     /* Verde sálvia */
--taipei-brown: #8F7368    /* Marrom */
--taipei-beige: #CDB5A7    /* Bege */
```

### Animações
- Fade in ao scroll (viewport triggers)
- Hover effects em cards e botões
- Transições suaves entre estados
- Animações de background nos heroes
- Menu mobile com slide animation

## 📦 Instalação

```bash
npm install
```

## 🛠️ Desenvolvimento

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## 🏗️ Build

```bash
npm run build
npm start
```
